Visual Studio

地球上最好用的 C++ 开发环境，没有之一。现在也有免费的社区版了，可惜以前密集写 C++ 代码的时候没赶上。

快捷键

功能 快捷键
快速打开 anything C-,
当前打开文件 C-M-Down
切换头/源文件 C-ko
返回 C–
前进 C-S–
跳转到定义 F12
在新窗口展示定义 M-F12
跳转到声明 C-M-F12
上/下移动行 M-Up/Down
在上面插入行 C-Enter
在下面插入行 C-S-Enter
折叠/展开当前代码段 C-mm
注释 C-kc
取消注释 C-ku
选中单词大写 C-S-u
选中单词小写 C-u
调试 F5
执行 C-F5
设置断点 F9
步过 F10
步入 F11

插件

  • Go To Definition

    Make ctrl+click perform a “Go To Definition” on the identifier under the cursor.

  • Visual Assist

    Productivity tool for C/C++ and C# that improves IDE features related to navigation, refactoring, code generation, and coding assistance.
